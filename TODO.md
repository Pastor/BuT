## 1. Система типов

Необходимо скорректировать систему типов следующим образом: существуют только два базовых типа bit и float, остальные типы выводятся из них.
float является математическим типом и при генерации кода преобразуется соответственно в float - C, FLOAT - ST, в asm используются 
функции по работе с операндами с плавающей точкой. Из bit выводятся все остальные типы, как то u8 -> [8: bit], u16 -> [16: bit] и т.д.
соответственно при генерации кода они сводятся к типам генерируемого языка u8 -> unsigned char, u16 -> unsigned short и т.д. 
Типы, которые превышают размер базовых типов генерируемого языка приводятся к массивам с выравниванием по ближайшей границы простого типа.

Скорректируй исходный код в соответствии с этим правилом. Добавь описание в документацию. Напиши тесты на проверку работоспособности этого
требования. Напиши примеры. Добавь файл `std.but` который будет в себе содержать определения базовых и часто используемых типов 
(выведенных из основных) и помести его в отдельную директорию `include`. Используй этот файл как подключаемый в примерах 
которые используют выводные типы.

Все комментарии, тезисы, описания формируй на русском языке.
